“¿Cómo podemos descubrir lo que Pepe le está diciendo a su amigo si su mensaje está escrito con un código secreto?”.

Hoy vamos a trabajar como “detectives de códigos” para descifrar el mensaje oculto.

Actividad: El código secreto

Instrucciones: ¿Qué le está diciendo Pepe a su amigo? Observa las palabras clave, sustituye cada símbolo por una de las letras que forman cada palabra clave y tendrás el código secreto para descifrar lo que dice Pepe.
